	Плагины грузятся из папки plugins. Для каждого плагина необходимо создавать отдельную категорию (имя папки плагина с заглавной буквы).Также модифицирован класс View онессы для корректной подгрузки вьюх плагинов.

	При загрузке DOM страницы автоматически инициализируется объект Connector.
	За отправку запроса отвечает метод request. Он обращается к методу модели, в зависимости от переданных данных (processor - класс,action - метод). Возвращает ответ либо в формате JSON, либо чистый html (если в запрос передан флаг pure).
